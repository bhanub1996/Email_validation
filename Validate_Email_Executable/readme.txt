Steps to run the Email_Validation on a csv file
i. Download the Email_validation.exe file from this repo
ii. Place all the emails in the first column of the csv file
iii. Place the csv file in the same directory as the executable and name file as 1.csv
iv. Run the exe file, the output will also be reflected in a new csv file named as 2.csv
v. The Output files contains the Emails in the first Column and True/False in the second column - True for a Valid Email and False for an Invalid one
